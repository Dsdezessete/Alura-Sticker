Alura Stickers

Este é um projeto em Java que utiliza a API do IMDb para buscar os top 250 filmes e extrair os dados relevantes (título, poster e classificação). Em seguida, utiliza esses dados para criar figurinhas personalizadas de cada filme e salvar em arquivos PNG.

Como utilizar

Clone o repositório em sua máquina local.
Abra o projeto em sua IDE de preferência (recomendamos o VS Code com a extensão Java Extension Pack).
Certifique-se de que possui o Java Development Kit (JDK) instalado em sua máquina.
Execute a classe AluraStickers.java para buscar os top 250 filmes, criar as figurinhas e salvá-las em arquivos PNG.
Bibliotecas utilizadas.

Gson: biblioteca para trabalhar com JSON em Java.
HttpClient: biblioteca nativa do Java para fazer requisições HTTP.
Java Image I/O API: biblioteca nativa do Java para trabalhar com imagens.
